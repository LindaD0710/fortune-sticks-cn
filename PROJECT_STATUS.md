## 📝 注意事项

- **两个版本独立运行**：英文版（3000端口）和中文版（3001端口）可以同时运行
- **共享后端**：两个版本使用相同的 Supabase 数据库和 API
- **环境变量**：中文版需要独立的环境变量配置（如果需要不同的配置）
- **Git 仓库**：建议为中文版创建独立的 Git 仓库或分支
