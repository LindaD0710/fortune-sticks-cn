# Creem 支付系统集成指南

## 📋 需要的信息

在开始集成之前，我需要了解 Creem 的以下信息：

1. **API 文档地址**：Creem 的开发者文档 URL
2. **API Key**：如何获取 Creem API Key
3. **支付流程**：
   - 是客户端集成（前端直接调用）还是服务端集成（需要后端 API）
   - 支付成功后如何接收回调（Webhook 或 Redirect）
4. **支付金额**：当前设置为 $1.99，是否需要可配置
5. **支付货币**：支持哪些货币

## 🔧 集成方案

### 方案 A：如果 Creem 提供前端 SDK

1. 安装 Creem SDK
2. 在支付按钮中调用 Creem API
3. 处理支付成功回调
4. 跳转到解读页面

### 方案 B：如果 Creem 需要服务端集成

1. 创建支付 API 路由 (`/api/payment/create`)
2. 创建支付回调 API 路由 (`/api/payment/callback`)
3. 在 Supabase 中存储支付记录
4. 验证支付状态后解锁解读
